# Home work 15 (Pizza project)


## Основные задания

### 1
Выводить информацию о пицце в модальном окне при клике на карточку с пиццей. Сделать возможность закрытия модального окна. Создать приятный дизайн. 

### 2
Добавить кнопку  "Создать пиццу" в header. При нажатии на нее вывести модальное окно где будет возможность ввести название пиццы, выбрать с помощью checkbox - ов ингридиенты и по нажатию на кнопку "Создать", которая будет находится в модальном окне, создать пиццу из введенных данных и поместить в pizzaList. У созданной пиццы должно быть свойство selfCreated: true. И на накрточке должно быть указанно что она созданна вами. Создать возможность закрытия модального окна.


## Доп.задания 

### 3
Добавить возможность при создании пиццы загружать изображение с пиццей.
Изображение должно быть в формате base64. Читаем [FileReader](https://developer.mozilla.org/ru/docs/Web/API/FileReader)
